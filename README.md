# @dinehq/react-seo

## Installation

```
pnpm add @dinehq/react-seo
```

```
npm install @dinehq/react-seo
```

```
yarn add @dinehq/react-seo
```

## Usage

```jsx
import { SEO } from '@dinehq/react-seo'

function App() {
  return <SEO description='This is a description' title='My App' />
}
```
